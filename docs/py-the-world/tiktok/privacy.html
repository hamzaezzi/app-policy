Privacy Policy for Py-The-World - TikTok Video Publisher

Py-The-World - TikTok Video Publisher ("the Application") respects user privacy.

This application does not collect, store, or process personal data.

The application only uses TikTok authentication to publish video content to a connected TikTok account using the official TikTok API.

Py-The-World - TikTok Video Publisher does not store personal information, user data, or video content.

No personal information is shared with third parties.

Access tokens provided by TikTok are used only for the purpose of publishing video content to the authorized TikTok account.
